# practice-dags

В этом репозитории собраны мои учебные DAG'и для оркестрации задач.  
Проекты показывают, как я осваивала построение пайплайнов и улучшала подход к их организации.

---

## 🚀 DAG "my_first_dag" — Первая попытка

Мой первый DAG.  
Здесь я разбиралась с базовой логикой построения пайплайна, зависимостями между задачами и структурой файла. 
Задача: с определенного сайта, собирающего статистику по популярности доменов сети, выгрузить данные и провести несколько аналитических действий: 

- Найти топ-10 доменных зон по численности доменов;
- Найти домен(ы) с самым длинным именем;
- Найти место, на котором находится домен airflow.com.

Что реализовано:
- Простейшая последовательность задач;
- Базовые операторы;
- Понимание принципа работы DAG;
- Понимание базовых параметров DAG.

> Этот DAG — отправная точка моего знакомства с планированием задач.

## 🔧 DAG "my_first_dag_modified" — Модифицированная версия первого проекта

Улучшенная версия первого DAG c другой бизнес-задачей: с помощью данного датасета по играм (http://kaggle.com/datasets/gregorut/videogamesales), ответить на вопросы: 

- Какая игра была самой продаваемой в этом году во всем мире?
- Игры какого жанра были самыми продаваемыми в Европе? 
- На какой платформе было больше всего игр, которые продались более чем миллионным тиражом в Северной Америке?
- У какого издателя самые высокие средние продажи в Японии?
- Сколько игр продались лучше в Европе, чем в Японии?
> Год был задан 2009-й

Что изменилось:
- Чтение сразу датасета;
- Добавлены новые задачи;
- Передача разных данных (датасет, массив, число);
- Использование более новой версии Airflow.

> Этот DAG показывает мой рост в понимании построения работы с DAG.
